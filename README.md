# Developer Blog

A responsive and modern developer blog built using **HTML, CSS, and JavaScript**. This project was created to provide a clean platform for sharing programming notes, technical articles, and learning resources.

## Features

* Responsive design for desktop and mobile devices
* Clean and modern user interface
* Blog-style article layout
* Smooth navigation and user-friendly experience
* Built using pure HTML, CSS, and JavaScript
* Fast loading with no external frameworks

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
